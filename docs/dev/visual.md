# 视觉设计

本应用以 [Holo 设计][adchs]为主，[Material Design 1][mdui-design] 作为补充。

## 应用主题

应用主题应该跟随系统。

可以考虑支持 EMUI、MagicUI 主题

## 系统图标

Holo 主题使用 Holo 图标，Material 主题使用 Material 图标。

姜饼主题的标签栏使用 Material 图标，其他复用 Holo 图标。
